## 1. 结论摘要

| 维度 | v1 决策 |
|---|---|
| 端 | 仅 Web；小程序管理端不做百科 |
| 菜单 | 内容管理 → 百科 |
| 权限 | 整站超级管理员 |
| 状态 | 草稿 / 公开 / 需解锁 / 下架；任意互转 |
| 对外公开内容 | 仅状态 = 公开 |
| 需解锁 | 仅后台标记；v1 不做解锁发放 |
| 下架 | 软不可见；数据保留；无硬删 |
| 城市 | 必选 cityId（下拉 = 启用运营城） |
| GPS | 公开时必填；存 WGS84；地图选点 + 数字可改；不校验城界；后台不转坐标系 |
| 媒体 | 0–9 张图；JPG/PNG/WebP；单张 ≤5MB；首张封面；无视频；无外链图 |
| 正文 | 完整 Markdown 字符串；后台不校验语法 |
| 标签 | 自由多标签；无标签库 |
| 页面 | 列表 + 新建/编辑同一表单；无预览页 |
| 列表 | 城/状态/标签筛选；标题+标签搜索；分页；行内改状态；编辑 |
| 排序 | 更新时间倒序；无运营权重 |
| 统计 | 只读浏览量、收藏量；后台不可重置 |
| 浏览量累计 | 详情成功打开 +1（含未登录） |
| 日志 | 创建 / 编辑 / 状态变更 |

---

## 2. 状态机

| 状态 | 说明 |
|---|---|
| 草稿 | 编辑中 |
| 公开 | 已上架；唯一对外公开态 |
| 需解锁 | 标记态；v1 无解锁链路，不发放、不写解锁记录 |
| 下架 | 撤回；数据与 ID 保留 |

- 四态任意互转  
- 下架：二次确认  
- 切到公开：跑公开校验，失败则不改状态并提示  
- 新建默认：草稿  

---

## 3. 实体模型

| 字段 | 说明 |
|---|---|
| 百科ID | 系统生成 |
| 标题 | ≤50；不要求唯一 |
| 所属城市 cityId | 必选；来自启用运营城 |
| GPS 经度 / 纬度 | WGS84；合法经纬度 |
| 正文 | Markdown 原文；≤20000 |
| 图片列表 | 0–9；有序；首张 = 封面 |
| 标签 | ≤10 个；每标签 ≤12 字 |
| 状态 | 四态 |
| 浏览量 / 收藏量 | 只读累计 |
| 创建人 / 更新人、时间 | 审计 |

---

## 4. 校验

### 4.1 草稿 / 需解锁 / 下架 保存

最低：标题（长度合法）+ 所属城市（启用城）。  
图片、正文、GPS、标签可缺；若填则校格式与长度。

### 4.2 公开

| 项 | 规则 |
|---|---|
| 标题 | 非空，≤50 |
| 所属城市 | 启用城 |
| GPS | 合法经纬度 |
| 正文 | 非空，≤20000 |
| 图片 | 可选，0–9 |
| 标签 | 可选 |

### 4.3 标签

- 去首尾空格；同词条去重；大小写保留；空丢弃  

### 4.4 图片

- 本地上传 JPG/PNG/WebP；单张 ≤5MB；可排序  

### 4.5 GPS

- 地图选点与经纬度数字双向同步；存 WGS84  

### 4.6 正文

- 完整 Markdown 原文存储；后台不做语法校验  

---

## 5. 与运营城市

- 新建/改绑：下拉仅启用城  
- 改城名不影响 cityId  
- 城停用后：已绑百科保留 cityId、状态不自动变；列表仍可按该城筛；不可再选该停用城为新所属；当前已绑停用城时可改其他字段，改城则只能选启用城  
- 不因城停用批量下架百科  
- 城市规则见 [后台管理-运营城市v1.md](./后台管理-运营城市v1.md)  

---

## 6. 页面规格

### 6.1 列表

- 筛：城市、状态、标签  
- 搜：标题 + 标签（不搜正文）  
- 分页；默认更新时间倒序  
- 展示：标题、城、状态、封面缩略、浏览/收藏、更新时间等  
- 行内：状态下拉；下架确认；公开失败提示原因  
- 新建、进入编辑；空态 + 新建  
- 不做：批量改状态、手动权重  

### 6.2 新建 / 编辑

- 同一表单；全部可写字段 + 状态  
- 浏览量/收藏量只读  
- 按目标状态执行 4.1 或 4.2  
- 无独立只读详情页、无预览页  

### 6.3 权限

- 菜单：内容管理 → 百科  
- 超管全开  

---

## 7. 统计与日志

- 浏览量：详情成功打开 +1（含未登录）；后台只读、不可重置  
- 收藏量：只读累计  
- 操作日志：创建、编辑、状态变更（操作人、对象、前后关键字段、时间）  

---

## 8. 并发与导入

- 后写覆盖；无乐观锁  
- 不做批量导入

## 9. 用例

| ID | 名称 | 前置 | 主成功路径 | 异常 | 后置 |
|---|---|---|---|---|---|
| BAIKE-UC-01 | 新建草稿 | ≥1 启用城 | 标题+城 → 存草稿 | 无启用城无法选城 | 状态=草稿 |
| BAIKE-UC-02 | 公开 | 草稿有标题+城 | 补 GPS+正文 → 公开 | 缺项失败不改态 | 状态=公开 |
| BAIKE-UC-03 | 列表公开失败 | 缺 GPS | 行内改公开 | 提示原因 | 状态不变 |
| BAIKE-UC-04 | 下架 | 公开 | 下架 → 确认 | 取消则不变 | 状态=下架 |
| BAIKE-UC-05 | 需解锁 | 任意 | 改需解锁保存 | 无发放入口 | 后台可筛 |
| BAIKE-UC-06 | 图集 | 编辑中 | 上传调序 | 超限失败 | 首张封面 |
| BAIKE-UC-07 | 标签 | 重复/空格 | 保存 | 规范化 | 标签干净 |
| BAIKE-UC-08 | 停用城百科 | 已绑停用城 | 改标题保存 | 不可改绑停用城 | 可改绑启用城 |

## 10. 验收

- 标题+启用城可存草稿；缺 GPS/正文不能公开  
- 公开/下架（含确认）/需解锁状态行为正确  
- 无解锁发放入口  
- 图 0–9、格式大小、首张封面  
- 标题可重名；标签限制生效  
- 搜标题与标签，不搜正文  
- 统计只读；日志可追溯  
- 所属城下拉仅启用城  

---

## 11. 相对总 PRD 收窄

| 总 PRD | 本专题 v1 |
|---|---|
| 公开 / 需解锁 / 草稿 | + 下架 |
| 图片 / 视频 | 仅多图 |
| 需解锁 + 解锁记录 | 仅标记，无解锁链路 |
| GPS | WGS84，后台不转换 |
| 用户投稿 | 仍不做 |

---

## 12. v1 不做

- 用户投稿、解锁发放/撤销  
- 视频、外链图、硬删  
- 批量导入、批量改状态、运营排序权重  
- 预览页、独立标签库  
- 正文搜索、乐观锁  
- 百科专属账号
